Scholarship Finder App is a desktop application developed using Java and Java Swing. The purpose of this project is to help students easily find scholarships based on their eligibility criteria such as age, education level, family income, and state.

The application provides a simple graphical user interface where users can enter their details and instantly see the scholarships they are eligible for. This system reduces the effort of searching through multiple websites for scholarship information.
Objectives

Help students quickly find suitable scholarships

Filter scholarships based on eligibility criteria

Provide a simple and easy-to-use interface

Demonstrate Java GUI development using Swing
Features

Simple graphical user interface

Input fields for age, education, income, and state

Scholarship eligibility filtering

Display of eligible scholarships with amount

Supports multiple states and scholarship programs
Technologies Used

Java

Java Swing

VS Code

Java Collections (ArrayList)
How It Works

User enters personal details:

Age

Education

Income

State

The system checks the scholarship database.

If the user's details match the eligibility criteria, the scholarship will be displayed.

The results appear in the results section.
